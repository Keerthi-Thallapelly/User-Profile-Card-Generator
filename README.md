# User-Profile-Card-Generator


A full-stack web application that allows users to enter their name, bio, and profile image URL to generate a dynamic profile card.

## Features

* User-friendly form interface
* Dynamic profile card generation
* REST API using Express.js
* Frontend and backend integration
* Responsive UI design

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js

## How to Run

1. Clone the repository
2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
npm start
```

4. Open index.html in your browser
